This manual will guide you through running the Werewolf game using the provided code, including game launch, role setup, gameplay flow, and core operation rules. It is suitable for both new and experienced players.

I. Basic Game Information

1.1 Game Features

  • Supports 25 Werewolf game roles (including Good and Werewolf factions, such as Seer, Witch, Werewolf King, Gargoyle, etc.)
  • Dual role assignment modes: Manual Role Assignment / Automatic Random Assignment by the System
  • Complete day-night cycle mechanism to restore the authentic Werewolf gameplay
  • Dual operation methods: Button click + Manual number input, with intuitive operation
  • Traditional Chinese interface with full prompts for role skills and rules

1.2 Execution Method

  1. Copy the code into a text editor (e.g., Notepad, VS Code) and save it as an HTML file (e.g., WerewolfGame.html).
  2. Open the file with any browser (Chrome, Edge, Firefox, etc.) to launch the game.

II. Game Operation Flow

2.1 Step 1: Select Role Assignment Mode

After launching the game, you will first enter the "Role Assignment Mode Selection" phase:

  • Click [1]: Manual Role Assignment Mode (fixed role list order; the Nth role corresponds to Player N)
  • Click [2]: Automatic System Assignment Mode (set the role list first, then the system shuffles and randomly assigns roles to players)

2.2 Step 2: Set Basic Game Information

  1. Set the number of players:
    • Number buttons (1~20) will be displayed on the screen. Click the total number of players for the game (at least 4 players are recommended).
    • You can also directly enter the number in the number input box and press Enter to confirm.
  2. Set the role pool:
    • All 25 roles will be listed on the screen (marked with ğŸŸ¢Good Faction / ğŸ”´Werewolf Faction + skill descriptions).
    • Select roles for "Role Pool Slot 1 to Slot N" in sequence (N = number of players). The system will confirm the role pool list after selection.

2.3 Step 3: Role Assignment (Automatic / Manual)

Mode 1: Automatic System Assignment

  • The system will automatically shuffle the set role pool, randomly assign roles to all players, and display "Player X â†’ Role (Faction)".
  • After assignment, the player numbers of the Good and Werewolf factions will be displayed automatically.

Mode 2: Manual Role Assignment

  • The system will show the remaining roles to be assigned. You need to select roles for "Player 1 to Player N" one by one (only from the role pool).
  • After assignment, the player numbers of the Good and Werewolf factions will be displayed automatically.

2.4 Step 4: Enter the Core Gameplay (Day-Night Cycle)

The core of the game is a cycle of "Night Action â†’ Day Vote â†’ Victory Judgment" until one faction wins.

ğŸ”¹ Night Phase (Role Skill Execution)

The system will wake up each role in sequence to execute their skills. The operation rules are as follows:

  1. Werewolf Faction: Choose a target to kill (click [0] to skip killing, resulting in a peaceful night). The Evil Knight cannot kill themselves.
  2. Guard: Choose a target to protect (cannot protect the same target two nights in a row; click [0] to skip).
  3. Dreamer: Must choose a target to hypnotize (no skip option). Hypnotizing the same target two nights in a row will kill them.
  4. Seer / Medium: Choose a target to inspect (click [0] to skip). The Seer checks the faction; the Medium checks the real identity.
  5. Witch: Can choose to "save the killed player (Antidote)", "poison a player (Poison)", or "skip (0)". Cannot use both potions in the same night.
  6. Demon Hunter: Unlocks hunting ability from the second night. Hunting a Werewolf kills the target; hunting a Good player kills themselves (click [0] to skip).
  7. Gargoyle: Wakes up alone to inspect identities. Can kill alone if all regular Werewolves are eliminated.
  8. Cupid: Must choose 2 players to be lovers (if one dies, the other dies of love).
  9. Other Roles: Execute corresponding skills according to prompts (e.g., Werewolf Brother, Magician, Gravekeeper).

End of Night: The system announces the players who died last night, triggering additional rules such as the Hunter shooting, lovers dying of love, and the Werewolf King taking a player down.

ğŸ”¹ Day Phase (Debate + Voting)

  1. Knight Duel: If there is a surviving Knight, they can choose to initiate a duel. Dueling a Werewolf kills the Werewolf and ends the day; dueling a Good player kills the Knight.
  2. Public Vote:
    • Surviving players vote in sequence to eliminate a target. You can click the player number or enter the number and press Enter.
    • Clicking [0] triggers a "Werewolf Self-Explosion Inquiry":
      • Click [1]: Select the Werewolf to self-explode. Self-explosion forces the day to end and enters the night (White Werewolf King / Werewolf King can take a player down when self-exploding).
      • Click [0]: No Werewolf self-explodes; continue voting.
  3. Voting Result:
    • The player with the most votes is eliminated (the Fool does not die when eliminated by vote, only loses voting rights).
    • A tie results in no elimination, and the game proceeds to the next day-night cycle.

2.5 Step 5: Victory Judgment and Restart

  1. Victory Conditions:
    • ğŸŸ¢ Good Faction: Win by eliminating all Werewolves.
    • ğŸ”´ Werewolf Faction: Win by eliminating all Good players.
  2. End of Game:
    • The system will display all players' real identities, survival status, voting rights, and skill status.
    • Choose [1] to restart the game (retains the role pool, can reassign roles) or [0] to end the game.

III. Core Operation Tips

3.1 Operation Methods

  • Button Click: Directly click the number/option buttons on the screen to confirm operations.
  • Manual Number Input: Enter the corresponding number (e.g., player number, option number) in the top input box and press Enter to confirm quickly. Suitable for experienced players.
